Create an IFC Building.

<p align="center">
  <img src="/img/ifc_building.png" />
</p>

## Input
1. **Model***: the IfcOpenShell model.

2. **Relating Object Id***: the relating object id (usually it's an IFC Site).

3. **Name**: the name of the building (default is "Hopper Building").

\* Required

## Output
1. **Model**: the IfcOpenShell model.

2. **Building Id**: the building id.
