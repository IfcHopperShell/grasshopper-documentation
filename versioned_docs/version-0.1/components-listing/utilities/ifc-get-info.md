Get the attributes of an IFC entity from its step id.

<p align="center">
  <img src="/grasshopper-documentation/img/ifc_get_info.png" />
</p>

## Input
1. **Model***: the IfcOpenShell model.

2. **Id***: the step id to look for.

\* Required

## Output
1. **Keyes**: the attributes keyes.

2. **Values**: the attributes values.
