Write an IFC file in step format.

<p align="center">
  <img src="https://ifchoppershell.github.io/grasshopper-documentation/img/ifc_write.PNG" />
</p>

## Input

1. **Model***: the IfcOpenShell model.

2. **Folder**: the folder where to save the file (default is an "IfcHopperShell" folder in the user root).

3. **File Name**: the file name (default is "Hopper File").

4. **Write**: a boolean to trigger file write. Use a button (default is False).

\* required

## Output

1. **File Path**: the full path where the file is saved to.
