Create empty IfcOpenShell model.

<p align="center">
  <img src="/img/ifc_model.png" />
</p>

## Input
1. **Version**: the version of IFC. Can be "IFC2X3", "IFC4" or "IFC4X3" (default is "IFC4").

2. **Name**: the name of the model (default is "Hopper Model").

3. **Description**: the description of the model (default is "...").

4. **Author Name**: the author name of the model (default is "...").

5. **Author Surname**: the author surname of the model (default is "...").

6. **Organization**: the organization authring of the model (default is "...").

7. **Authorization**: the manager responsible for the model (default is "...").

## Output
1. **Model**: the IfcOpenShell model.
