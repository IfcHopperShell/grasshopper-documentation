Create an IFC Site.

<p align="center">
  <img src="https://ifchoppershell.github.io/grasshopper-documentation/img/ifc_site.PNG" />
</p>

## Input
1. **Model***: the IfcOpenShell model.

2. **Relating Object Id***: the relating object id (usually an Ifc Project).

3. **Name**: the site name (Default is "Hopper Site").

\* Required

## Output
1. **Model**: the IfcOpenShell model.

2. **Site Id**: the site id.
