Create an IFC Building Storey.

<p align="center">
  <img src="/grasshopper-documentation/img/ifc_building_storey.png" />
</p>

## Input
1. **Model***: the IfcOpenShell model.

2. **Relating Object Id***: the relating object id (usually an Ifc Building).

3. **Name**: the project name (default is "Hopper Project").

\* Required

## Output
1. **Model**: the IfcOpenShell model.

2. **Storey Id**: the building storey id.
